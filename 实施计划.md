### 1.2 协议层 `pkg/protocol`

#### 1.2.1 `types.go` — 常量与类型码

```go
package protocol

// FrameType 帧类型
type FrameType byte

const (
    FrameTunnelOpen  FrameType = 0x01
    FrameTunnelData  FrameType = 0x02
    FrameTunnelClose FrameType = 0x03
    FramePing        FrameType = 0x04
    FramePong        FrameType = 0x05
)

const (
    FrameHeaderSize = 41      // 1(type) + 36(connID) + 4(payloadLen)
    UUIDSize        = 36      // "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
)

// 预定义的连接 ID 常量
var (
    ZeroConnID = "00000000-0000-0000-0000-000000000000"
)
```

#### 1.2.2 `frame.go` — 帧结构与编解码

**Frame 结构体：**

```go
type Frame struct {
    Type      FrameType
    ConnID    string   // 36 字节 UUID 字符串
    Payload   []byte
}

func (f *Frame) Encode() ([]byte, error)
func DecodeFrame(raw []byte) (*Frame, error)
func ReadFrame(conn *websocket.Conn) (*Frame, error)
func WriteFrame(conn *websocket.Conn, f *Frame) error
```

**实现要点：**

| 函数 | 关键逻辑 |
|------|----------|
| `Encode` | 组装 41 字节头 + Payload；ConnID 不足 36 字节补零；PayloadLen 大端写入 |
| `DecodeFrame` | 校验最小长度 41；校验 ConnID 格式；校验类型码范围 |
| `ReadFrame` | 读取 WS 二进制消息 → `DecodeFrame` |
| `WriteFrame` | `Encode` → 写入 WS 二进制消息 |

**边界处理：**
- 收到非法帧类型 → 返回 `ErrUnknownFrameType`，调用方断开连接
- PayloadLen 超过限制（默认 4MB）→ 返回 `ErrPayloadTooLarge`
- 空 Payload 允许（TUNNEL_CLOSE、PING 等）

#### 1.2.3 `frame_test.go` — 帧编解码测试

| 测试用例 | 覆盖 |
|----------|------|
| Encode/Decode 往返（所有帧类型） | 确认编码后解码一致 |
| 边界：零长度 Payload | TUNNEL_CLOSE 场景 |
| 边界：大 Payload（1MB） | 确认编码解码正确 |
| 异常：非法类型码 | 期望错误 |
| 异常：截断数据 | 期望错误 |
| 异常：ConnID 格式错误 | 期望错误 |

**证据：** `go test ./pkg/protocol/ -v` 全绿

### 2.2 服务入口 `pkg/server/server.go`

```
Start(config)
  │
  ├── 初始化 TunnelManager
  │
  ├── 注册 HTTP 路由:
  │   ├── GET /_tunnel/   → handleTunnelUpgrade (WS 升级 + 认证)
  │   └── 任意其他路径     → handleBrowserRequest (泛域名路由 + 中继)
  │
  └── http.ListenAndServe(addr, handler)
```

**`handleTunnelUpgrade` 逻辑：**

```
收到 HTTP 请求 → 路径为 /_tunnel/
  │
  ├── 路径不匹配 → http.NotFound
  │
  ├── 检查 Upgrade: websocket 头
  │   └── 非 WS 升级请求 → 返回 400
  │
  ├── 调用 Authenticate(r) → 从 Authorization 头提取 Bearer Token
  │   ├── Token 为空或不匹配 → 401 + "invalid token"
  │   └── Token 匹配 → 继续
  │
  ├── 从 X-Tunnel-Subdomains 头解析子域名列表（逗号分隔）
  │   ├── 逐个检查子域名是否已存在路由表中
  │   │   └── 已存在 → 409 + "subdomain xxx taken"
  │   └── 全部未占用 → 继续
  │
  ├── 执行 websocket.Upgrade(w, r, nil) → 101
  │
  └── 创建 TunnelSession:
      生成 session ID (UUID)
      设置 Subdomains / Token / Conn
      注册到 TunnelManager (所有子域名 → 该 session)
      启动 handleSession(session) goroutine
```

**`handleBrowserRequest` 逻辑：**

```
收到 HTTP 请求
  │
  ├── 解析 Host → 提取子域名
  │   ├── 子域名空或等于主域名 → 404
  │   └── 提取成功 → 查 TunnelManager
  │       ├── 未找到 → 404
  │       └── 找到 TunnelSession
  │           ├── 检查 session 是否存活
  │           │   └── 已断开 → 502
  │           └── 存活 → hijack TCP 连接 → NewRelay
```

### 2.3 认证 `pkg/server/auth.go`

func Authenticate(r *http.Request, globalToken string) error

### 2.4 隧道管理器 `pkg/server/manager.go`

```go
type TunnelManager struct {
    mu          sync.RWMutex
    tunnels     map[string]*TunnelSession  // subdomain → session
    globalToken string
}

func NewTunnelManager(globalToken string) *TunnelManager
func (m *TunnelManager) Register(session *TunnelSession) error    // 注册所有子域名
func (m *TunnelManager) Unregister(session *TunnelSession)        // 注销所有子域名
func (m *TunnelManager) Lookup(subdomain string) *TunnelSession   // 查找

```

**实现要点：**

- `Lookup`：读锁，快速查找
- 所有子域名操作必须持有锁

### 2.5 隧道会话 `pkg/server/session.go`

```go
type TunnelSession struct {
    ID          string
    Subdomains  []string
    Token       string
    Conn        *websocket.Conn
    manager     *TunnelManager
    activeConns map[string]context.CancelFunc
    lastPong    time.Time
    mu          sync.Mutex
    done        chan struct{}
}

func NewTunnelSession(conn *websocket.Conn, subdomains []string, token string, manager *TunnelManager) *TunnelSession
func (s *TunnelSession) Start(ctx context.Context)     // 启动帧读取循环 + 心跳监控
func (s *TunnelSession) Stop()                         // 停止 + 清理
func (s *TunnelSession) HandleFrame(frame *protocol.Frame) error
```

**帧读取循环 `Start`：**

```
for {
    frame, err := protocol.ReadFrame(s.Conn)
    if err != nil → 断开清理
    switch frame.Type {
    case protocol.FramePing:
        回复 Pong (ZeroConnID, 空载荷)
    case protocol.FramePong:
        s.lastPong = time.Now()
    case protocol.FrameTunnelClose:
        关闭对应 conn_id 的活跃连接
    case protocol.FrameTunnelData:
        查找 activeConns[conn_id] → 写入浏览器 TCP
    default:
        log.Warn("unknown frame type")
    }
}
```

**心跳监控（在 Start 中启动一个 ticker goroutine）：**

```
ticker = time.NewTicker(interval)
for {
    select {
    case <-ticker.C:
        if time.Since(s.lastPong) > interval * timeoutMultiplier {
            log.Warn("heartbeat timeout")
            s.Stop()
        }
    case <-s.done:
        return
    }
}
```

### 2.6 请求中继 `pkg/server/relay.go`

```go
type Relay struct {
    TunnelConnID string
    BrowserConn  net.Conn
    Session      *TunnelSession
    cancel       context.CancelFunc
}

func NewRelay(browserConn net.Conn, session *TunnelSession) *Relay
func (r *Relay) Start(ctx context.Context)
func (r *Relay) handleBrowserToTunnel(ctx context.Context)
func (r *Relay) handleTunnelToBrowser(ctx context.Context)
```

**双向数据拷贝：**

| Goroutine | 方向 | 操作 |
|-----------|------|------|
| A: browser→tunnel | 浏览器 TCP → WS | 循环 `browserConn.Read(buf)` → `protocol.WriteFrame(session.Conn, TUNNEL_DATA{connID, buf})` |
| B: tunnel→browser | WS → 浏览器 TCP | 收到 connID 匹配的 TUNNEL_DATA → `browserConn.Write(payload)` |

**清理逻辑：**
- 任一 goroutine 退出 → `cancel()` → 两个 goroutine 都退出
- 发送 `TUNNEL_CLOSE` 通知对方
- 从 session.activeConns 移除
- 关闭 `browserConn`

### 3.2 客户端入口 `pkg/client/client.go`

**`Run` 生命周期：**

```
Run(ctx)
  │
  └── reconnectLoop(ctx):
      attempt = 0
      loop:
        err = connect(ctx)
        if err != nil {
            if IsAuthError(err) {
                log.Fatal("认证失败: ", err)   // 认证失败不重试
            }
            sleep(min(60, 2^attempt))
            attempt++
            goto loop
        }
        attempt = 0
        session.Wait()  // 阻塞直到断开
        goto loop       // 非认证断开 → 重连
```

**`connect` 逻辑：**

```
解析 ws:// 或 wss:// 地址
构建 HTTP 头:
  Authorization: Bearer <token>
  X-Tunnel-Subdomains: <comma-separated keys from Proxies>

发起 WebSocket 升级:
  ├── HTTP 401 → 返回 ErrAuthFailed("invalid token")
  ├── HTTP 409 → 返回 ErrSubdomainConflict("subdomain xxx taken")
  ├── HTTP 101 → 创建 TunnelSession, 返回 nil
  └── 其他错误 → 返回 连接错误（可重试）
```

}

**帧循环 `Start`：**

```
go sendHeartbeat(ctx, s)  // 每间隔发送 PING

for {
    frame, err := protocol.ReadFrame(s.Conn)
    if err != nil → s.Stop(); return
    switch frame.Type {
    case protocol.FrameTunnelOpen:
        subdomain := extractSubdomain(frame.Payload)
        targetAddr, ok := s.Proxies[subdomain]
        if !ok {
            // 未知子域名，拒绝
            protocol.WriteFrame(s.Conn, &Frame{Type: FrameTunnelClose, ConnID: frame.ConnID})
            continue
        }
        localConn, err := net.Dial("tcp", targetAddr)
        if err != nil {
            protocol.WriteFrame(s.Conn, &Frame{Type: FrameTunnelClose, ConnID: frame.ConnID})
            continue
        }
        fwd := NewLocalForward(frame.ConnID, localConn, s)
        s.addForward(fwd)
        go fwd.Start(ctx)
    case protocol.FrameTunnelData:
        fwd := s.getForward(frame.ConnID)
        if fwd != nil {
            fwd.LocalConn.Write(frame.Payload)
        }
    case protocol.FrameTunnelClose:
        fwd := s.getForward(frame.ConnID)
        if fwd != nil {
            fwd.Close()
            s.removeForward(frame.ConnID)
        }
    case protocol.FramePong:
        // 更新最后 PONG 时间（可选统计）
    }
}
```

**心跳发送 (goroutine)：**

```
ticker := time.NewTicker(interval)
for {
    select {
    case <-ticker.C:
        protocol.WriteFrame(s.Conn, &Frame{Type: FramePing, ConnID: ZeroConnID})
    case <-s.done:
        return
    }
}
```
